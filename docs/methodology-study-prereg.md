# 预注册分析计划：命理流派受控对比研究（v1，锁定于数据采集前）

**锁定时间**：2026-07-10（本文件提交后开始跑数，跑数后不得修改假设与判读规则；任何偏离须在论文中披露）

## 1. 研究问题
在统一底模、统一排盘基座、统一作答协议下，不同命理方法论层对 BaziQA 客观题准确率的因果贡献是什么？命理方法论是否优于（a）无方法论、（b）纯统计基率、（c）近零信息量对照（称骨）？

## 2. 设计
- **数据**：BaziQA contest8_2021–2025 全量 200 题（90 命主）。无 dev/holdout 切分（本研究无调参、无变体选择，全部臂在同一数据上单次运行）。
- **底模**：deepseek/deepseek-v4-pro（OpenRouter），temperature=0.3，max_tokens=16384，非流式，同一解析器，解析失败重试至多 3 次，仍失败按 0 分计。
- **九臂**：
  | 臂 | 系统提示词 | 注入件 |
  |---|---|---|
  | A0 bare | 通用分析师 | 仅公历生辰（地板） |
  | A1 chart | 无方法论 | +精确排盘 |
  | A2 ziping | 子平格局法(~1.1k字) | +排盘 |
  | A3 mangpai | 生产盲派提示词(4.9k字) | +排盘+K线摘要 |
  | A4 ziwei | 紫微断法(~1k字) | +排盘+紫微盘 |
  | A5 minglv | 命律体系(~1.2k字) | +排盘+实律数机算判决 |
  | A6 tieban | 铁板条文法(~0.6k字) | +排盘+起数条文（python引擎） |
  | A7 bayes | 纯统计学家（明示禁用命理） | +排盘（安慰剂）+人口统计基率 |
  | A8 chenggu | 称骨法(~0.5k字) | +排盘+骨重批断（近零信息对照） |

## 3. 预先声明的假设（跑数前写死）
- **H1（排盘效应）**：A1 − A0 ≥ +4pp（基于前研究 +6.4pp 的先验）。
- **H2（方法论增量）**：各方法论臂（A2–A6、A8）相对 A1 的增量彼此不可区分，|Δ| ≤ 5pp。
- **H3（统计对命理）**：A7 不劣于方法论臂的中位数（差值 > −3pp）。
- **H4（低信息对照）**：A8 称骨与 A1 差值在 ±5pp 内（若某大流派显著优于称骨 ≥7pp，构成该流派携带信息的初步证据）。

## 4. 统计方案
- 主指标：200 题准确率；95% CI 用 Wilson 区间。
- 成对比较：同题配对 McNemar 精确检验；比较族 = {各方法论臂 vs A1} 共 7 个，Holm 校正；A1 vs A0、A7 vs 流派中位数为独立的两个预设检验。
- **判读铁律**：|Δ| < 7pp 一律报告为"不可区分"（n=200 配对功效所限）；题型切片（年份题/家境题等）仅作探索性分析并明确标注。
- 拒答/解析失败计 0 分并单独报告作答率；某臂引擎故障（如铁板起数失败的命主）按提示词内声明的降级路径作答，不重跑其他臂。

## 5. 已知混淆（事先披露）
① 提示词长度不均（盲派 4.9k 字系生产资产原样保留，其余 0.5–1.2k）；② 各臂专属注入件信息量不同（该差异是"各派自带工具箱"设计的一部分，非缺陷，但解释时须区分"方法论"与"工具箱"贡献）；③ 单次运行、temperature 0.3 非 0，存在采样噪声；④ 称骨骨重表丙辰一格取通行本值；⑤ 基准答案本身可能含少量标注误差。

## 6. 赛前预判（供赛后对照，检验研究者先验校准度）
排盘效应 +4~8pp；六派彼此挤在 ±5pp 噪声带；A7 贝叶斯落在派系区间上半部；称骨与 A1 打平。若实际结果偏离以上任何一条，将在论文中如实报告并讨论。
